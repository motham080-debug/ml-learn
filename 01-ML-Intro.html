ประเภทของ Machine Learning

Machine Learning สามารถแบ่งออกเป็น 3 ประเภทหลักตามวิธีการเรียนรู้

Supervised Learning

การเรียนรู้แบบมีผู้สอน: เรียนรู้จากข้อมูลที่มีคำตอบ (Label) กำกับไว้แล้ว

Unsupervised Learning

การเรียนรู้แบบไม่มีผู้สอน: เรียนรู้จากข้อมูลที่ไม่มีคำตอบกำกับ

Reinforcement Learning

การเรียนรู้แบบเสริมแรง: เรียนรู้จากการลองผิดลองถูกและรางวัล

Supervised Learning (การเรียนรู้แบบมีผู้สอน)

อัลกอริทึมเรียนรู้จากข้อมูลที่มีคำตอบ (Label) กำกับไว้แล้ว เพื่อทำนายผลลัพธ์สำหรับข้อมูลใหม่

Linear Regression

ใช้ทำนายค่าตัวเลขต่อเนื่อง โดยสร้างสมการความสัมพันธ์เชิงเส้นตรงระหว่างตัวแปรต้นและตัวแปรตาม

การใช้งาน: ทำนายราคาบ้าน, ยอดขายรายปี, อุณหภูมิเฉลี่ย

sklearn.linear_model อ่านคู่มือ Scikit-Learn →

Logistic Regression

ใช้จำแนกประเภท (Classification) ทำนายความน่าจะเป็นของข้อมูลที่จะเกิดในแต่ละคลาส (Binary หรือ Multiclass)

การใช้งาน: ตรวจจับอีเมลสแปม, การทำนายการอนุมัติสินเชื่อ

sklearn.linear_model อ่านคู่มือ Scikit-Learn →

Decision Trees

จำลองการตัดสินใจเป็นแผนภาพต้นไม้ เข้าใจง่ายและถอดกฎเกณฑ์ทางธุรกิจ (IF-THEN) ออกมาได้ดี

การใช้งาน: การวินิจฉัยโรคทางการแพทย์, การแบ่งกลุ่มข้อมูลลูกค้า

sklearn.tree อ่านคู่มือ Scikit-Learn →

SVM

หาเส้นแบ่งระนาบสูงสุด (Hyperplane) เพื่อแยกประเภทข้อมูล และรองรับขอบเขตที่คดเคี้ยวด้วยเทคนิค Kernel

การใช้งาน: การรู้จำใบหน้า, การจัดกลุ่มภาพทิวทัศน์, การวิเคราะห์ไบโอเซนเซอร์

sklearn.svm อ่านคู่มือ Scikit-Learn →

k-NN

เปรียบเทียบข้อมูลใหม่กับข้อมูลที่อยู่ข้างเคียงที่มีระยะห่างสั้นที่สุด k ตัว และตัดสินผลตามเสียงส่วนใหญ่

การใช้งาน: ระบบวิเคราะห์หาลูกค้าที่ใกล้เคียง, ระบบแนะนำเพลง/ภาพยนตร์

sklearn.neighbors อ่านคู่มือ Scikit-Learn →

Naive Bayes

คำนวณตามหลักความน่าจะเป็นของเบย์ (Bayes' Theorem) ประสิทธิภาพสูงมากกับการวิเคราะห์ข้อความและทำงานได้เร็ว

การใช้งาน: คัดกรองอีเมลสแปม, จัดกลุ่มข่าว, การจำแนกอารมณ์ความรู้สึกเชิงบวก/ลบ

sklearn.naive_bayes อ่านคู่มือ Scikit-Learn →

Random Forest

โมเดลรวมกลุ่ม (Ensemble Model) ที่ผสานต้นไม้ตัดสินใจจำนวนมากมาร่วมคำนวณเพื่อลด Overfitting และเพิ่มความเสถียร

การใช้งาน: ประเมินความเสี่ยงทางการเงิน, การวิเคราะห์ข้อมูลทางการแพทย์

sklearn.ensemble อ่านคู่มือ Scikit-Learn →

Unsupervised Learning (การเรียนรู้แบบไม่มีผู้สอน)

อัลกอริทึมเรียนรู้จากข้อมูลที่ไม่มีคำตอบกำกับ เพื่อค้นหาโครงสร้างหรือรูปแบบที่ซ่อนอยู่ในข้อมูล

K-Means Clustering

จับกลุ่มข้อมูลออกเป็น k กลุ่ม โดยหาจุดศูนย์กลางที่ลดระยะทางเฉลี่ยระหว่างข้อมูลกับจุดศูนย์กลางของกลุ่มนั้น ๆ

การใช้งาน: แบ่งเซกเมนต์ลูกค้าตามยอดช้อปปิ้ง, แบ่งกลุ่มพฤติกรรมผู้เล่นเกม

sklearn.cluster อ่านคู่มือ Scikit-Learn →

DBSCAN

แบ่งกลุ่มตามความหนาแน่นของจุดข้อมูล ข้อดีคือแบ่งรูปร่างของกลุ่มข้อมูลที่ทับซ้อนกันซับซ้อนได้ และคัดกรอง Noise อัตโนมัติ

การใช้งาน: แบ่งพื้นที่ภูมิศาสตร์, จัดกลุ่มภาพอิงตามความหนาแน่นของเม็ดสี

sklearn.cluster อ่านคู่มือ Scikit-Learn →

PCA

ลดมิติข้อมูลแบบเป็นเส้นตรง โดยสร้างแกนมิติใหม่ที่อธิบายความแปรปรวน (Variance) ของข้อมูลได้สูงที่สุด

การใช้งาน: ลดขนาดตัวแปรภาพถ่าย, เตรียมข้อมูลก่อนเข้า Neural Network

sklearn.decomposition อ่านคู่มือ Scikit-Learn →

t-SNE

ลดมิติข้อมูลแบบไม่เป็นเส้นตรง โดยให้จุดข้อมูลที่อยู่ใกล้กันในมิติสูง ยังคงอยู่ใกล้กันเมื่อลดมิติเหลือ 2D หรือ 3D

การใช้งาน: พล็อตโครงสร้างโมเดลความสัมพันธ์คำศัพท์ (Word Embeddings), แผนภาพจำแนกกลุ่มแบบ Visualization

sklearn.manifold อ่านคู่มือ Scikit-Learn →

Isolation Forest

ตรวจจับความผิดปกติ (Anomaly Detection) โดยใช้สุ่มโครงสร้างต้นไม้แยกข้อมูลที่แปลกแยกออกมา โดยจะแยกเดี่ยวได้เร็วที่สุด

การใช้งาน: การตรวจจับบัตรเครดิตโดนรูดใช้งานที่น่าสงสัย, การคัดกรองขยะสัญญาณเซนเซอร์

sklearn.ensemble อ่านคู่มือ Scikit-Learn →

Reinforcement Learning (การเรียนรู้แบบเสริมแรง)

อัลกอริทึมเรียนรู้จากการลองผิดลองถูกและรางวัล โดยตัวแทน (Agent) จะเรียนรู้ว่าควรทำการกระทำใดในสถานการณ์ต่างๆ เพื่อให้ได้รับรางวัลสูงสุด

Q-Learning

เรียนรู้ค่าคุณภาพ (Q-value) ของการกระทำในแต่ละสถานการณ์ เพื่อหานโยบาย (Policy) ที่ดีที่สุด

ตัวอย่างการใช้งาน: การควบคุมหุ่นยนต์, การเล่นเกม, การจัดการทรัพยากร

Deep Q-Network (DQN)

ใช้ Deep Neural Network ในการประมาณค่า Q-value สำหรับสถานการณ์ที่มีความซับซ้อนและมีมิติสูง

ตัวอย่างการใช้งาน: การเล่นเกมที่ซับซ้อน, การควบคุมหุ่นยนต์ในสภาพแวดล้อมจริง, การขับรถอัตโนมัติ

Policy Gradient

เรียนรู้นโยบาย (Policy) โดยตรง โดยปรับปรุงพารามิเตอร์ของนโยบายเพื่อเพิ่มผลตอบแทนที่คาดหวัง

ตัวอย่างการใช้งาน: การควบคุมหุ่นยนต์ที่มีการเคลื่อนไหวต่อเนื่อง, การเล่นเกมที่มีพื้นที่การกระทำต่อเนื่อง

Actor-Critic

ผสมผสานระหว่าง Value-based และ Policy-based โดยมีส่วน Actor ที่เรียนรู้นโยบาย และส่วน Critic ที่ประเมินค่าของสถานการณ์

ตัวอย่างการใช้งาน: การควบคุมหุ่นยนต์ที่ซับซ้อน, การเล่นเกมที่มีพื้นที่สถานะและการกระทำขนาดใหญ่

เปรียบเทียบประเภทของ Machine Learning

ประเภท ข้อมูลที่ใช้ เป้าหมาย ตัวอย่างอัลกอริทึม
Supervised Learning ข้อมูลที่มีคำตอบ (Label) กำกับ ทำนายผลลัพธ์สำหรับข้อมูลใหม่ Linear Regression, Logistic Regression, Decision Trees, SVM, Neural Networks
Unsupervised Learning ข้อมูลที่ไม่มีคำตอบกำกับ ค้นหาโครงสร้างหรือรูปแบบที่ซ่อนอยู่ในข้อมูล K-Means Clustering, Hierarchical Clustering, PCA, Anomaly Detection
Reinforcement Learning ข้อมูลจากการลองผิดลองถูกและรางวัล เรียนรู้นโยบายที่ดีที่สุดเพื่อให้ได้รับรางวัลสูงสุด Q-Learning, Deep Q-Network, Policy Gradient, Actor-Critic
